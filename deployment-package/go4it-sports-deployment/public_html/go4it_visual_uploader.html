Go4It Visual Uploader & Deployment Tool

This powerful tool allows for seamless file transfers between your Replit environment and your server, with a beautiful visual interface and advanced integration with Monaco Editor + StarCoder AI.

Getting Started

To use this tool, simply include the JavaScript file in your project and initialize it. The uploader will create a full-screen interface for managing files and deployments.

<script src="go4it_visual_uploader.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    // Initialize the uploader
    window.Go4ItVisualUploader.init();
  });
</script>

Key Features

  • Visual File Management - Side-by-side view of local and remote files
  • Drag & Drop Uploads - Easy file transfers with progress tracking
  • Deployment Pipeline - Visual step-by-step deployment process
  • Integration Status - Real-time monitoring of connected services
  • Guided Tour - Interactive onboarding for new users
  • Real-time Logging - Detailed logs of all operations
  • Monaco Editor Integration - Seamless connection with Monaco + StarCoder

Configuration Options

You can customize the uploader by modifying the CONFIG object at the top of the JavaScript file:

const CONFIG = {
  SERVER_URL: "http://188.245.209.124",  // Your server URL
  FILE_BROWSER_API: "/api/files",        // File Browser API endpoint
  ADMIN_USERNAME: "admin",               // File Browser admin username
  DEPLOYMENT_PATH: "/var/www/go4itsports", // Deployment path on server
  MONACO_PATH: "/var/www/html/pharaoh",  // Monaco Editor path on server
  STARCODER_API_URL: "http://localhost:11434/v1", // StarCoder API URL
  // Theme colors can also be customized
};
