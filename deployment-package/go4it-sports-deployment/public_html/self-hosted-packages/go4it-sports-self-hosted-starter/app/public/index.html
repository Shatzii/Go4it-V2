Go4It Sports File Transfer Utility

This utility helps you transfer the Go4It Sports client files from Replit to your server.

Step 1: Download the Client Files

Download the client files as a ZIP archive. This package contains all the necessary files for your website's frontend.

Step 2: Upload to Your Server

After downloading the files, you need to upload them to your server at the correct location.

Option A: Using SSH/SCP

If you have SSH access to your server, use the following SCP command:

scp -r /path/to/downloaded/zip username@your-server:/var/www/go4itsports/
cd /var/www/go4itsports/
unzip client_deployment.zip -d client/dist/
chown -R www-data:www-data client/dist/
chmod -R 755 client/dist/

Option B: Using File Browser

If you have File Browser installed on your server:

  1. Access File Browser at your server address (e.g., http://your-server-ip:8080)
  2. Log in with your credentials
  3. Navigate to /var/www/go4itsports/
  4. Upload the ZIP file
  5. Extract the ZIP file to client/dist/
  6. Set proper permissions (owner: www-data, mode: 755)

Option C: Using cPanel or Similar

If your server has cPanel or a similar control panel:

  1. Access the File Manager in your control panel
  2. Navigate to your website's root directory
  3. Upload the ZIP file
  4. Extract to the appropriate directory

Step 3: Verify Installation

After uploading and extracting the files, verify that your website is working correctly:

  1. Visit your website in a browser (http://go4itsports.org)
  2. Check that all pages load correctly
  3. Verify that the website can communicate with the API

If everything is working correctly, your deployment is complete!
