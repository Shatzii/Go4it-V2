File Structure

All HTML files are stored in the server/public directory. The clean URL middleware maps the URL paths to their corresponding HTML files.

Key Middleware

The server/middleware/clean-urls.ts file handles mapping clean URLs to their corresponding HTML files.

API Endpoints

  • /api/auth - Authentication endpoints
  • /api/users - User management endpoints
  • /api/schools - School management endpoints
  • /api/courses - Course management endpoints
  • /api/analytics - Analytics endpoints

AI Integration Endpoints (Active)

  • /api/ai/integration/status - Check AI services status
  • /api/ai/integration/create-teacher - Create an AI teacher system prompt
  • /api/ai/integration/teacher-response - Generate a response from an AI teacher
  • /api/ai/integration/learning-plan - Generate a learning plan for a student
  • /api/ai/integration/curriculum-content - Generate curriculum content
  • /api/ai/integration/learning-style-assessment - Assess a student's learning style
