Using Parent Translation Mode

Parent Translation Mode helps bridge language barriers by showing content in both English and your selected language. This makes it easier for non-native speaking parents to support their children's learning.

How It Works

Parent Translation Mode displays all text content in two languages simultaneously:

Original Content

The original English content appears in standard text with a subtle blue line on the left side.

Translations

The translation in your selected language appears below the original text in blue italic font with a darker blue line.

Navigation Elements

For buttons, links, and navigation elements, translations appear more compactly next to the original to save space.

Getting Started

Choose Your Language

First, select your preferred language from the language selector in the header (look for the globe icon).

Enable Parent Mode

After selecting your language, check the "Parent Translation Mode" checkbox in the language selector dropdown.

Browse the Content

As you browse the site, all content will now be displayed in both English and your selected language.

Toggle As Needed

You can easily turn Parent Mode on or off at any time using the language selector.

Tip

For the best experience, set your language in the language selector first, then enable Parent Translation Mode.

Frequently Asked Questions

How many languages are supported?

Currently, Parent Translation Mode supports German and Spanish in addition to English. We plan to add more languages in the future.

Will this affect my child's view of the content?

No. Parent Mode is designed for parents only. Your child can continue to view the content in the language that best suits their learning needs.

Can I use Parent Mode with accessibility features?

Yes! Parent Mode is fully compatible with all of our accessibility features including dyslexia-friendly mode, high contrast mode, and other settings.

Are translations perfect?

Our translations are professionally created, but no translation is perfect. If you find any issues with translations, please let us know so we can improve them.

Can I save my Parent Mode preference?

Yes, your Parent Mode preference is saved automatically and will be remembered the next time you visit the site.
