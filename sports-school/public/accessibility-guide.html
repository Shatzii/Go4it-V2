Neurodivergent Accessibility Features

Our platform is designed with accessibility at its core, offering specialized features for dyslexia, ADHD, autism spectrum, and other neurodivergent learning styles.

Overview

ShatziiOS is built from the ground up to support neurodivergent learners. Our accessibility features go beyond standard compliance to provide a truly inclusive learning environment where every student can thrive.

All accessibility options can be accessed through:

  • Accessibility Panel - Click the accessibility icon in the bottom right corner of any page
  • Keyboard Shortcuts - Quick access to features using keyboard combinations
  • User Profile Settings - Save your preferences for consistent experience across sessions

Your accessibility settings are remembered across sessions and automatically applied when you return to the platform.

Dyslexia-Friendly Features

Our dyslexia support is designed to make reading and writing more accessible through specialized fonts, color adjustments, and spacing controls.

OpenDyslexic Font

A typeface specifically designed to increase readability for readers with dyslexia, with weighted bottoms to indicate direction and unique letter shapes.

Letter Spacing

Adjustable spacing between letters reduces the "crowding effect" that many dyslexic readers experience, making it easier to distinguish between similar letters.

Reading Ruler

A horizontal guide that follows your cursor to help maintain focus on the current line of text and reduce getting lost in dense paragraphs.

Line Height Control

Increased space between lines reduces visual stress and helps prevent skipping lines or losing place while reading longer texts.

The OpenDyslexic font helps many readers by providing letters with heavier weighted bottoms. This gives the letters a sense of gravity, helping to prevent them from appearing to float or flip. The unique letter shapes also help prevent confusion between similar-looking letters like 'b' and 'd'.

ADHD-Friendly Features

Our ADHD-focused accommodations help with focus, minimize distractions, and provide clear structure to learning materials.

Focus Mode

Simplifies the interface by removing unnecessary elements and highlighting only the current content, reducing visual distractions.

Responsive Layout

Content is presented in digestible chunks with clear section breaks, making it easier to process information and maintain focus.

Break Reminders

Optional timers that suggest short breaks to prevent cognitive fatigue and help maintain optimal focus during longer learning sessions.

Reduced Animations

Minimizes motion on screen to reduce distractions while preserving essential interactive elements.

Autism-Friendly Features

Our autism accommodations focus on predictability, clear navigation, and sensory considerations.

Color Adjustment

Softer color palettes and reduced contrast options that minimize visual stress and sensory overload while maintaining readability.

Animation Suppression

Automatically disables animations and transitions that can be distracting or overwhelming for users with sensory processing differences.

Clear Structure

Consistent, predictable layout with clear headings and navigation to ensure users always know where they are and what to expect.

Notification Control

Fine-grained control over system notifications and alerts, allowing users to manage potential interruptions and sensory input.

High Contrast Mode

Our high contrast mode provides maximum readability for users with visual processing differences or low vision.

High Contrast Demo

High Contrast Example

The high contrast mode uses:

  • Black background with white text for maximum contrast
  • Bold, clear borders around interactive elements
  • Enhanced focus indicators for keyboard navigation
  • Increased text size options

This mode makes content more accessible for users with contrast sensitivity or certain visual processing differences.

Keyboard Shortcuts

Quick access to accessibility features using keyboard shortcuts:

Shortcut Action
Alt + D Toggle Dyslexia-Friendly Mode
Alt + C Toggle High Contrast Mode
Alt + A Toggle ADHD-Friendly Mode
Alt + U Toggle Autism-Friendly Mode
Alt + + Increase Font Size
Alt + - Decrease Font Size
Alt + R Toggle Reading Ruler
Alt + M Toggle Reduced Motion
Alt + 0 Reset All Settings
