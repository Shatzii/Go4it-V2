API Documentation

The Go4It Sports platform provides RESTful APIs for video upload, analysis, and highlight generation.

Base URL

/api

Available Endpoints

  • GET /api/health - Check API health
  • GET /api/videos?userId=:id - Get videos for user
  • GET /api/videos/:id - Get video by ID
  • GET /api/highlights?videoId=:id - Get highlights for video
  • GET /api/highlights?featured=true&limit=10 - Get featured highlights
  • GET /api/highlight-configs - Get highlight generator configurations
  • GET /api/highlight-configs/active - Get active highlight configurations
  • POST /api/videos/upload - Upload a new video
  • POST /api/videos/:id/generate-highlights - Generate highlights for video
  • POST /api/highlights/generate-title - Generate title for highlights collection
