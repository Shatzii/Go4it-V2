# GAR Fundamentals

Overview

An introduction to the Game-Athletic Rating (GAR) system used by Go4it coaches. Covers how scores are calculated, what the 0–100 scale means, and how to interpret a 0–5 star translation.

Course Outline

1. What GAR measures and why it matters.
2. Basic movement & skill tests used for scoring.
3. Mapping GAR to development plans.
4. Using GAR to set measurable goals.

How to use

Take the tests with a coach or by following the instructions in this guide, then upload results for a coach review.
